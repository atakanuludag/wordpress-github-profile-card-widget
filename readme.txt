Shows your github profile in detail.


== Description ==
Widget The Github Profile Card plugin allows you to show your github profile in detail.

== Installation ==
1. Upload `widget-github-profile-card` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to wordpress admin panel to Appearance > Widget.
4. Activate the Github Profile Card.

* English
